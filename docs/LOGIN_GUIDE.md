# 登录功能说明

## 已完成的修改

### 1. 集成扫码登录到主界面 ✅

- 删除了独立的 `accounts.html` 页面
- 删除了重复的登录模态框
- 将二维码扫码登录集成到主界面的"登录小红书"按钮

### 2. 使用方法

1. **打开主界面**：`http://localhost:3000`

2. **点击"登录小红书"按钮**：
   - 弹出登录模态框
   - 自动加载二维码

3. **扫码登录**：
   - 使用小红书 APP 扫描二维码
   - **重要：不要关闭浏览器窗口！**
   - 系统每 2 秒自动检查登录状态
   - 登录成功后自动关闭模态框

4. **如果二维码过期**：
   - 点击"刷新二维码"按钮

### 3. 重要提示

⚠️ **扫码后不要关闭浏览器窗口！**

- 系统需要通过浏览器窗口检测登录状态
- 如果关闭窗口，系统无法检测到登录成功
- 请等待系统自动检测并提示"登录成功"

### 4. 账户管理

- 主界面左侧导航栏有"账号管理"按钮
- 点击进入账户管理页面
- 可以添加、切换、删除账户

### 5. 发布功能

- 发布时自动使用当前活跃账户
- 可以在账户管理页面切换账户
- 发布前会检查账户登录状态

## 故障排除

### 问题：扫码后没有反应

**原因**：关闭了浏览器窗口

**解决**：
1. 不要关闭浏览器窗口
2. 等待系统自动检测（每 2 秒检查一次）
3. 看到"登录成功"提示后再关闭

### 问题：二维码加载失败

**解决**：
1. 检查服务器是否正常运行
2. 点击"刷新二维码"按钮
3. 查看浏览器控制台的错误信息

### 问题：一直显示"等待登录"

**原因**：可能是网络问题或小红书服务异常

**解决**：
1. 刷新二维码重试
2. 检查网络连接
3. 查看服务器日志
